Vision

WCAG 2.0 Reference: 1.1 Text Alternatives

Check if TalkBack is installed

Make sure TalkBack is installed and configured

Having TalkBack installed is essential to the functioning of our accessible features for vision impaired users. 

            
/**
 * Checks to see that Google TalkBack is installed
 * @param context The context that wishes to know the results
 * @return true if TalkBack is installed
 */
public boolean isTalkBackInstalled(Context context);

/**
 * Checks to see that Google TalkBack is currently enabled
 * @param context The context that wishes to know the results
 * @return true if TalkBack is enabled
 */
public boolean isTalkBackEnabled(Context context);

/**
 * Checks to see that explore by touch (provided via TalkBack) is currently enabled
 * @param context The context that wishes to know the results
 * @return true if explore by touch is enabled
 */
public boolean isExploreByTouchEnabled(Context context);
WCAG 2.0 Reference: 4.1 Compatible

Optimal TalkBack Pitch and Speed

The user must have the ability to modify the pitch and speed of the talking voice

TalkBack must be fully customizable for the user, in order to attend to their needs. This includes the manipulation of TalkBack settings. 


/**
 * Provides an Intent that will open the Google Play store, at the TalkBack installation screen
 * @param context The context that wishes to know the results
 * @return The Intent that will open the TalkBack Play Store page
 */
public Intent getTalkBackPlayIntent(Context context);
